# Portfolio

This is an archive of projects done between the years 2021 and present day by me.

## About

This portfolio showcases the work of **Paridhi Garg**, a creative technologist and experiential designer currently based in New York. The portfolio features interactive projects that explore the intersections of human behavior, design, and technology. The projects are primarily designed digital applications and immersive experiences.

## Portfolio Overview

### Project Categories

The portfolio contains **14 projects** organized into four main categories:

- **Installations** - Interactive installations exploring quantum computing, human-machine interactions, and immersive experiences
- **AI Applications** - Web-based AI applications that think about the role AI plays in everyday life, using multimodal capabilities such as text, image and sound synthesis, image analysis, and object detection
- **3D Environments** - Narrative-driven 3D experiences and speculative futures created with Unity and Blender
- **Others** - Diverse projects including audiovisual work, games, and experimental installations

### Featured Projects

- **This Machine is a Stranger** (2025) - Exploring trust and mistrust in human-machine interactions
- **Fleeting States + Measured Values** (2024) - Interactive installation visualizing quantum computing concepts exhibited in Germany
- **Move A Bit** (2024) - Interactive installation capturing live motion and visualizing quantum entanglement and noise
- **Computerized Memories** (2023) - 3D environment exploring the biological and psychological structure of memory
- **Sights and Insights** (2025) - Voice-based AI application transforming ordinary walks into curious interventions
- **Granny Bytes** (2024) - AI application exploring intergenerational connections through technology

### Technologies & Tools

Projects utilize a wide range of technologies including:
- **UI/UX**: Figma, Adobe Creative Suite (AI, PSD, ID, PR, LR), HTML, CSS, JavaScript (node.js, next.js and express.js), Python
- **Creative Technologies**: TouchDesigner, Madmapper, Isadora, Physical Computing Sensors, LED Mapping
- **3D & Rendering**: Unity, Blender
- **AI/ML**: Machine Learning, StyleGANs, Text Generation, Image Generation, Prompt Engineering, Image Analysis, Sound Synthesis, text-to-speech · speech-to-text, Gesture Recognition · Object Detection
- **Fabrication**: Laser Cutting, 3D printing

### Key Themes

The work consistently explores:
- Human-computer interaction
- Memory, perception, and cognition
- Speculative futures and alternative realities
- Quantum computing and physics computing
- Environmental and spatial awareness
